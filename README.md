# Student Management API

A simple REST API built using **Node.js and Express.js** to manage student data.

## Features
- Get all students
- Get student by ID
- Search students by course
- Add new student
- Update student
- Delete student

## Technologies Used
- Node.js
- Express.js
- REST API

## API Endpoints

GET /students  
GET /students/:id  
GET /search?course=Node.js  
POST /students  
PUT /students/:id  
DELETE /students/:id  

## Run the Project

```bash
npm install
node server.js
